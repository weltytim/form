INSTALLATION

1. Reference the CSS and JS files in your <head>. For example:

<link rel="stylesheet" href="magicthumb.css" type="text/css" media="screen, projection"/>
<script type="text/javascript" src="magicthumb-packed.js"></script>

2. Set-up an image to enlarge by linking the small image to the big image. For example:

<a href="big.jpg" class="MagicThumb"><img src="small.jpg"/></a>

3. That's all!

4. To upgrade, buy Magic Thumb and overwrite magicthumb-packed.js with the full version.
