Magic Thumb™: slideshow example

This example shows how Magic Thumb™ can be used for a slideshow with keyboard navigation.

Check out the ultra-thin MacBook Air!

Right Arrow = next image 		Small is beautiful - the iPod Nano

Page Up = previous image Page Down = next image 		Apple MacBook

Spacebar = next image

Keyboard navigation:

  • Space bar - next image.
  • Escape - shrink image.
  • Page Up or Arrow Up or Arrow Left - previous image.
  • Page Down or Arrow Down or Arrow Right - next image.
